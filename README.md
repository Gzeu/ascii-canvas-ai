## ✨ Features

### 🎨 Advanced Drawing Tools
- **✏️ Freehand** - Smooth drawing with customizable characters
- **📏 Line** - Precise line drawing with multiple styles
- **▢ Rectangle** - Perfect rectangles with border styles
- **◆ Diamond** - Geometric diamond shapes
- **○ Circle** - Circular shapes with mathematical precision
- **➜ Arrow** - Directional arrows with custom heads
- **📝 Text** - Multi-line text placement
- **🪣 Fill** - Flood fill tool for area coloring
- **💨 Spray** - Artistic spray paint effect
- **🧹 Eraser** - Multiple size erasers
- **⬚ Select** - Smart selection with flood fill
- **✋ Move** - Move selected areas

### 🎯 Drawing Styles
- **Border Styles**: Simple, Double, Rounded, Heavy, Light
- **Arrow Heads**: Classic (`>`), Triangle (`▶`)
- **Line Styles**: ASCII, Light, Heavy, Dashed, Dotted
- **Diamond Styles**: Simple (`/`, `\`), Unicode (`◇`)
- **Freehand Characters**: `*`, `#`, `+`, `.`, `x`, `o`, `~`
- **Eraser Sizes**: 1×1, 3×3, 5×5

### ⚡ Advanced Features
- **Undo/Redo** (`Ctrl+Z` / `Ctrl+Y`) - Full history management
- **Clipboard** (`Ctrl+Shift+C`) - Copy ASCII to clipboard
- **Zoom** (scroll wheel) - 0.5× to 3× zoom
- **Pan** (Space+drag) - Smooth canvas navigation
- **Grid Background** - Visual alignment guide
- **Status Bar** - Real-time cursor and tool information
- **Keyboard Shortcuts** - Professional workflow
- **Smart Selection** - Flood-fill connected shapes
- **Dark Theme** - Easy on the eyes

### 🎮 User Experience
- **Immediate Drawing** - Freehand tool active on load
- **Visual Feedback** - Real-time tool status
- **Intuitive Interface** - Emoji-labeled tools
- **Responsive Design** - Works on all screen sizes
- **No Dependencies** - Single HTML file
- **Offline Ready** - Works without internet

## � Drawing Techniques

### Basic Shapes
```
Rectangle: Click and drag to create
Circle: Click center, drag for radius
Line: Click start, drag to end
Diamond: Click corner, drag to opposite
```

### Advanced Features
```
Fill Tool: Click area to fill with character
Spray Tool: Click and drag for artistic effect
Select Tool: Click connected areas to select
Move Tool: Select area, then move to new position
```

### Text Placement
```
- Click to position text cursor
- Enter text in prompt
- Support for multi-line text
- Automatic boundary checking
```

## 🏗️ Technical Architecture

### Core Technologies
- **Frontend**: Pure HTML5, CSS3, Vanilla JavaScript
- **Canvas**: HTML5 Canvas API for rendering
- **No Dependencies**: Self-contained single file
- **No Build Process**: Ready to use immediately

### Canvas System
- **Grid Size**: 200 columns × 80 rows
- **Font**: JetBrains Mono (12px)
- **Rendering**: Hardware-accelerated canvas
- **Zoom**: Smooth 0.5× to 3× scaling
- **Performance**: Optimized for real-time drawing

### Algorithms
- **Bresenham's Line Algorithm** - Perfect line drawing
- **Flood Fill Algorithm** - Smart area selection
- **Circle Generation** - Mathematical precision
- **Spray Pattern** - Randomized artistic effect
